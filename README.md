# Uskin Icons Font

## Development

### Install

grunt-webfont requires `fontforge` and `ttfautohint`.

```sh
# for macOS
brew install ttfautohint fontforge --with-python
```

Install `grunt` and `grunt-webfont`.

```sh
npm install
```

### Build

Bundles all svg icons into fonts.

```sh
npm run build
```
